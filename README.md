## 👋 About

I'm a **Forward Deployed Engineer at [TrueFoundry](https://www.truefoundry.com/)** (Bengaluru), helping customers turn LLM platforms into production systems that hold up under real traffic.

Before that I was an **ML Engineer at [Chubb](https://www.chubb.com/)**, shipping GenAI at scale — fine-tuning, agents, RAG, and high-throughput inference.

I also build open-source agent tooling (**[tracesage](https://github.com/kjgpta/tracesage)**) and publish NLP research on evaluation, multilingual systems, and code-switching.

## ⚡ Featured — tracesage

> Local-first observability for **LangChain & LangGraph**. Drop in one callback — watch every run as a live graph + timeline. No cloud account. No Docker. Featured in **[Python Weekly #750](https://www.pythonweekly.com/p/python-weekly-issue-750-june-18-2026)**.

```bash
pip install "tracesage[langchain]"
tracesage demo   # opens the live UI — no API key needed
```
